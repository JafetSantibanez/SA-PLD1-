# 🔢 Ordenador de Números

Aplicación web desarrollada con **HTML, CSS y JavaScript** que permite ingresar 3 números, analizarlos y mostrarlos ordenados de diferentes maneras.

---

## ✨ Funcionalidades

✅ Solicita 3 números al usuario  
✅ Detecta:

- Número mayor
- Número del centro
- Número menor

✅ Ordena los números:

- De menor a mayor
- De mayor a menor

✅ Detecta si:

- Los 3 números son iguales
- Existen 2 números iguales

✅ Interfaz visual amigable y responsiva

---

## 🖼️ Vista previa

### Pantalla principal
